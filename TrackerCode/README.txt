Building Capacity Tracker

This is the source code for the OSU Building Capacity Tracker.
To install this app on an Android Device, open it in AndroidStudio and install like normal.
The server should be up and running. 
To verify connection, make sure you see the words "SUCCESS" displayed in the test box at the bottom of the UI.
